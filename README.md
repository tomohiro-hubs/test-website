# Challenger GYM - 東村山・秋津の格闘技×パーソナルトレーニングジム

## プロジェクト概要

東村山・秋津エリアの格闘技ジム「Challenger GYM」の公式Webサイト（3ページ構成）です。
Breaking Down出場、YouTube登録者70万人の大黒力斗が代表を務める本格派ジムとして、無料体験（LINE予約・問い合わせ）の最大化を目的とし、初心者・女性・親子でも安心して通える格闘技ジムとしてのブランディングを実現しています。

**現行サイト**: https://challenger-gym.com/

### サイト構成（3ページ）

1. **index.html** - メインランディングページ（10セクション）
2. **classes.html** - クラス・料金詳細ページ（9セクション）
3. **kids.html** - キッズクラス（S×G School）専用ページ（9セクション）

## 主な特徴

### デザインコンセプト
- **カラースキーム**: 黒・白をベースに、アクセントカラーとして赤（格闘技の力強さ）を使用
- **トーンマナ**: 「怖くない格闘技ジム」として、清潔感と少人数制の安心感を演出
- **レスポンシブデザイン**: PC、タブレット、スマートフォン全デバイスに対応

### ターゲットユーザー
- 東村山・秋津/新秋津周辺在住の方
- 運動不足・ダイエットに悩む初心者
- 護身術を学びたい女性
- 子どもの習い事を探している親子
- ダンスに興味がある方
