# フローティングCTAボタン - 実装ガイド

## 📋 実装内容

スクロールに追従する「無料体験予約」ボタンを追加しました。

## 🎯 機能

### PC版（デスクトップ）
- **表示位置**: 画面右下
- **表示タイミング**: ヒーローセクションをスクロールで通過した後
- **非表示タイミング**: フッターに到達する前
- **アニメーション**: 
  - フェードイン/アウト
  - パルスアニメーション（注目を集める）
  - ホバー時に浮き上がる効果

### モバイル版
- **フローティングCTA**: 非表示（邪魔にならない）
- **下部固定バー**: 既存のモバイルCTAバーを使用
  - Instagram
  - LINEで無料体験

## 🎨 デザイン仕様

### ボタンデザイン
```
┌─────────────────┐
│      📱 LINE     │
│  無料体験予約     │
│ LINEでかんたん1分 │
└─────────────────┘
```

- **背景色**: 赤（--primary-color）
- **形状**: 丸みのあるピル型
- **影**: ソフトシャドウ
- **サイズ**: 最小幅180px

### アニメーション
1. **フェードイン**: 0.3秒
2. **パルス**: 2秒間隔で繰り返し
3. **ホバー**: 上に3px浮き上がる

## 🔧 カスタマイズ方法

### 表示タイミングを変更
```javascript
// js/main.js の以下の行を編集
if (currentScroll > heroHeight - 200) {  // 200を変更
    floatingCta.classList.add('visible');
}
```

### ボタンの位置を変更
```css
/* css/style.css */
.floating-cta {
    bottom: 30px;  /* 下からの距離 */
    right: 30px;   /* 右からの距離 */
}
```

### ボタンのテキストを変更
```html
<!-- index.html -->
<span class="floating-cta-text">無料体験予約</span>
<span class="floating-cta-subtext">LINEでかんたん1分</span>
```

### アニメーションを無効化
```css
/* css/style.css */
.floating-cta.visible .floating-cta-btn {
    /* animation: pulse 2s infinite; をコメントアウト */
}
```
